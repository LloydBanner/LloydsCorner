Prey Mooncrash (DLC)

Prey

Full Review

This game has so many great and creative elements. You have a gluegun which you can use to make bridges and traverse the map to places you wouldn't normally be able to reach, there are powers where you can mimic objects which allows you to hide but also to fit into small spaces, there are a bunch of creative weapons such as a crossbow that shoots foam darts that you can use to toggle control panels. All of these elements are really well thought out out and you could build your character around most of them. When I go back I intend to play completely differently than last time I played.

There are also a bunch of survival elements that I had turned off for the base game but are required in the DLC such as oxygen tanks in space which I think are probably worth playing the base game with.

My main issues with the game were the crafting system which didn't feel needed but didn't make the game worse and it being a little overwhelming at the start. Also the stealth systems don't really work so I would avoid them.

This game is great and absolutely worth playing. I feel like it went under the radar and I'm not too sure why because everything it does it does well and it does a lot.

Mooncrash

Full Review

I may have liked Prey's DLC more than the base game. I have replayed it a few times and not replayed the base game yet. It mixes Prey's core mechanics with an almost rougelike system. When you dye you play as a different character on the space station with different abilities. The goal is to escape with each character, but you gain more information and access to more items as you go. I think this is a great concept that uses the best elements of Prey to their best.
